# EAV-nodejs

#### Prerequisites

- Docker (18+)
- docker-compose (1.23+)
- NodeJS (10.0 +)
- npm (6+)

#### Using

Starting Postgres inside Docker

```bash
npm run docker:start
```

Making database migrations

```bash
npm run database:up
```

Starting server

```bash
npm run server:start
```
